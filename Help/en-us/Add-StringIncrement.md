# Add-StringIncrement

## SYNOPSIS
Increments a string

## SYNTAX

```
Add-StringIncrement [-String] <String> [<CommonParameters>]
```

## DESCRIPTION
Increments a given string "\<some text\> \<int\>" by one.

## EXAMPLES

### Connections

```powershell
PS C:\> Add-StringIncrement -String "Local Area Connection 1"
```

Returns "Local Area Connection 2"
