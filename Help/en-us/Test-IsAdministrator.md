# Test-IsAdministrator

## SYNOPSIS

Test if the current user is a local administrator

## SYNTAX

```
Test-IsAdministrator [<CommonParameters>]
```

## DESCRIPTION

Test if the current user is a local administrator

## EXAMPLES

### Example 1
```powershell
PS C:\> Test-IsAdministrator
```

Returns true or false
