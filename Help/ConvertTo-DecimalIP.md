# ConvertTo-DecimalIP

## SYNOPSIS
Converts a given IPv4 to decimal

## SYNTAX

```
ConvertTo-DecimalIP [-IPAddress] <IPAddress> [<CommonParameters>]
```

## DESCRIPTION
Converts a given IPv4 to decimal

## EXAMPLES

### Convert an IP

```powershell
PS C:\> ConvertTo-DecimalIP -IPAddress 192.168.2.101
```

Returns 3232236133
